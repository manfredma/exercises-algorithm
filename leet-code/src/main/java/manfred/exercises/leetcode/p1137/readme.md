# 第 N 个泰波那契数

泰波那契序列 `Tn` 定义如下：

```
T0 = 0, T1 = 1, T2 = 1
Tn+3 = Tn + Tn+1 + Tn+2（n >= 0）
```

给你整数 `n`，请返回第 `n` 个泰波那契数 `Tn` 的值。

## 示例 1：

```
输入：n = 4
输出：4
解释：
T3 = 0 + 1 + 1 = 2
T4 = 1 + 1 + 2 = 4
```

## 示例 2：

```
输入：n = 25
输出：1389537
```

## 提示：

- `0 <= n <= 37`
- 答案保证是一个 32 位整数，即 `answer <= 2^31 - 1`
