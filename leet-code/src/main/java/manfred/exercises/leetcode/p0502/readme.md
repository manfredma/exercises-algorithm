# IPO

假设力扣即将开始 IPO。为了以更高的价格将股票卖给风险投资公司，力扣希望在 IPO 之前开展一些项目以增加其资本。由于资源有限，它只能在 IPO 之前完成最多 `k` 个不同的项目。

给你 `n` 个项目，对于每个项目 `i`：

- `profits[i]`：完成该项目获得的纯利润
- `capital[i]`：启动该项目需要的最小资本

最初你的资本为 `w`，完成一个项目后利润会添加到总资本中。
从给定项目中选择**最多 `k` 个不同项目**，以**最大化最终资本**。

> 答案保证在 32 位有符号整数范围内。

## 示例 1：

```
输入：k = 2, w = 0, profits = [1,2,3], capital = [0,1,1]
输出：4
解释：
  初始资本为 0，只能启动 0 号项目（capital=0）。
  完成后资本变为 0+1=1，此时可以启动 1 号或 2 号项目。
  选择 2 号项目（profit=3），最终资本 = 0+1+3 = 4。
```

## 示例 2：

```
输入：k = 3, w = 0, profits = [1,2,3], capital = [0,1,2]
输出：6
```

## 提示：

- `1 <= k <= 10^5`
- `0 <= w <= 10^9`
- `n == profits.length == capital.length`
- `1 <= n <= 10^5`
- `0 <= profits[i] <= 10^4`
- `0 <= capital[i] <= 10^9`
