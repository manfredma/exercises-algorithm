# 可回收且低脂的产品

> 本题为 SQL 题,无 Java 方法签名。解法写在 `Solution.sql`。

表:`Products`

| 列名 | 类型 |
|---|---|
| `product_id` | int |
| `low_fats` | enum |
| `recyclable` | enum |

`product_id` 是该表的主键(具有唯一值的列)。
`low_fats` 是枚举类型,取值为 `('Y', 'N')`,其中 `'Y'` 表示该产品是
低脂产品,`'N'` 表示不是低脂产品。
`recyclable` 是枚举类型,取值为 `('Y', 'N')`,其中 `'Y'` 表示该产品
可回收,而 `'N'` 表示不可回收。

编写解决方案找出既是低脂又是可回收的产品编号。

返回结果无顺序要求。

## 示例 1：

```
输入：
Products 表：
+-------------+----------+------------+
| product_id  | low_fats | recyclable |
+-------------+----------+------------+
| 0           | Y        | N          |
| 1           | Y        | Y          |
| 2           | N        | Y          |
| 3           | Y        | Y          |
| 4           | N        | N          |
+-------------+----------+------------+

输出：
+-------------+
| product_id  |
+-------------+
| 1           |
| 3           |
+-------------+
解释：只有产品 id 为 1 和 3 的产品,既是低脂又是可回收的产品。
```
